Sorts

The subdirectories herein contain the sorts from the book and a harness (common/test_sort.cpp). that both tests and times them.

Errata

C++11 was still under development when the sorts were written. Though the codes compiled and ran with the Intel 12.1 compiler and VS2010 headers, later we discovered that const-qualified lvalues should not be used with std::move. This issue has been corrected in the listings, so you'll see fewer const qualifiers than in the book. The book listings not using std::move should nonetheless work with C++98 compilers.
