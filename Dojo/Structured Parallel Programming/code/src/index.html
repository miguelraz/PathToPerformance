Note on File Organization

Be aware that the usage of headers files differs from typical use of header files:
  • To preserve line numbers from the book, many of the files do not #include prerequisites.
  • Files that would ordinarily be *.cpp files are *.h files, so that they can be included by the harness. The harnesses deal with including prerequisites.
Comments are sparse since the examples are explained in the book.
